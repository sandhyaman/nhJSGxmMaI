# Python图书馆管理系统 python1131

## 项目概述

本项目是基于Python语言的图书馆管理系统，采用前后端分离的设计模式，前端使用Vue.js框架，后端使用Django框架，数据库采用MySQL。

## 技术栈

- 前端：Vue.js
- 后端：Django Python
- 数据库：MySQL

## 功能模块

系统主要包括以下功能模块：

- 首页
- 个人中心
- 用户管理
- 图书馆管理
- 图书信息管理
- 图书类型管理
- 借阅信息管理
- 归还信息管理
- 续借信息管理
- 罚金信息管理
- 留言板管理
- 系统管理

## 系统角色

- 系统管理员：负责系统整体管理和维护
- 普通用户：可以进行图书查询、借阅、归还等操作

## 运行环境

- 操作系统：Linux或Windows
- Python版本：3.6及以上
- MySQL版本：5.7及以上

## 部署步骤

1. 安装Python环境
2. 安装MySQL数据库，并创建相应数据库和表结构
3. 克隆后端代码到本地，安装依赖，配置数据库连接信息
4. 克隆前端代码到本地，安装依赖
5. 前后端分别启动，运行项目

## 目录结构

```
├── frontend  # 前端代码目录
│   ├── node_modules
│   ├── public
│   ├── src
│   └── vue.config.js
└── backend  # 后端代码目录
    ├── apps
    ├── db
    ├── libs
    ├── manage.py
    └── settings.py
```

## 核心亮点

- 前后端分离，易于维护和扩展
- 使用成熟的技术栈，保证系统稳定性和可维护性
- 功能完善，覆盖图书馆管理的主要业务需求
- 支持多角色操作，满足不同用户的使用需求
- 代码结构清晰，方便二次开发和定制化需求实现
